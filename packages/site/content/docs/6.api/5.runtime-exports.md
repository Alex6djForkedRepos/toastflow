# Runtime Exports

Toastflow exposes the same runtime model through each package. Use the import path that matches your app, then keep the same API names across Vue, Nuxt, and headless usage.

::callout{icon="i-tabler-file-code-2" color="primary"}
**Full TypeScript definitions:** [`packages/core/src/types.ts`](https://github.com/adrianjanocko/toastflow/blob/main/packages/core/src/types.ts){target="\_blank"} and [`packages/vue/src/index.ts`](https://github.com/adrianjanocko/toastflow/blob/main/packages/vue/src/index.ts){target="\_blank"}.
::

## Import Paths

| App type           | Import from              | Use for                                     |
| ------------------ | ------------------------ | ------------------------------------------- |
| Vue app            | `vue-toastflow`          | Plugin, components, `toast`, types, helpers |
| Nuxt auto-imports  | `nuxt-toastflow` module  | `toast`, `useToast()`, `<ToastContainer />` |
| Nuxt explicit path | `nuxt-toastflow/runtime` | Manual component/API imports                |
| Headless runtime   | `toastflow-core`         | Store, state, events, types, and utilities  |

## Package Map

::code-tree{default-value="vue/runtime.ts" expand-all}

```ts [vue/runtime.ts]
import {
  createToastflow,
  toast,
  ToastContainer,
  Toast,
  ToastProgress,
  type ToastConfig,
  type ToastState,
} from "vue-toastflow";
```

```ts [nuxt/auto-imports.ts]
// Auto-imported after modules: ["nuxt-toastflow"]
toast.success({ title: "Saved" });

const toastApi = useToast();
toastApi.dismissAll();
```

```ts [nuxt/runtime.ts]
import {
  toast,
  ToastContainer,
  Toast,
  ToastProgress,
  type ToastConfig,
  type ToastState,
} from "nuxt-toastflow/runtime";
```

```ts [headless/core.ts]
import {
  createToastStore,
  type ToastConfig,
  type ToastState,
  type ToastInstance,
} from "toastflow-core";
```

## Nuxt Runtime Path

Use `nuxt-toastflow/runtime` only when auto-imports are not enough:

::code-group

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["nuxt-toastflow"],
  toastflow: {
    componentName: false,
  },
});
```

```vue [app.vue]
<script setup lang="ts">
import { ToastContainer } from "nuxt-toastflow/runtime";
</script>

<template>
  <ToastContainer />
  <NuxtPage />
</template>
```

::

For normal Nuxt pages, prefer the module auto-imports documented in [toast vs useToast](/docs/nuxt/toast-and-use-toast).
